# RTAIP - Real-Time Analysis and Intelligence Platform

Welcome to the **RTAIP** repository! This project is a web application focused on real-time analysis and intelligence capabilities for military and tactical operations.

## Features
- **Real-Time Data Analysis**: Analyze data as it comes in.
- **MIL-STD Icon Integration**: Standardized military symbols for clarity.
- **ISR Asset Management**: Manage intelligence, surveillance, and reconnaissance assets.
- **Course of Action Comparisons**: Tools for analyzing different tactical options.
- **Single Page Application**: Smooth user experience with client-side routing.

## Getting Started

### Prerequisites
Make sure you have the following installed:
- [Node.js](https://nodejs.org/) (v14 or later)
- [Python](https://www.python.org/) (v3.7 or later)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/kamikaze1120/RTAIP.git
   cd RTAIP
   ```

2. Install frontend dependencies:
   ```bash
   cd frontend
   npm install
   ```

3. Install backend dependencies:
   ```bash
   cd backend
   pip install -r requirements.txt
   ```

4. Start the development servers:
   - Frontend:
     ```bash
     npm start
     ```
   - Backend:
     ```bash
     python app.py
     ```

## Usage
- Navigate to `http://localhost:3000` in your web browser to access the application.
- Explore the features and functionalities, including real-time data analysis and ISR asset management.
